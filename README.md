1. **F-Droid (Rekomendasi):** [Download Termux via F-Droid](https://f-droid.org/en/packages/com.termux/)
   *(Disarankan unduh aplikasi F-Droid terlebih dahulu, lalu cari 'Termux' di dalamnya agar lebih mudah update)*
2. **GitHub Resmi:** [Termux Releases v0.118+](https://github.com/termux/termux-app/releases)
   *(Pilih file .apk yang sesuai dengan arsitektur HP Anda, biasanya versi `universal` atau `arm64-v8a`)*
